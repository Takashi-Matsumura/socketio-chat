## Getting Started

git clone してください。

### ClientSide: Next.js 14

```bash
cd client
npm install
npm run dev
```

.env を作成して次の環境変数を設定してください。

```
NEXT_PUBLIC_SERVER_URL=http://localhost:5000
```

### ServerSide: Express

```bash
cd server
npm install
npm run start
```
